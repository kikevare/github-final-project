Simple Interest Calculator
This project implements a Simple Interest Calculator written in Python as part of the GitHub Final Project from IBM. The goal of this project is to demonstrate basic financial calculations while practicing essential Git and GitHub workflows such as branching, merging, pull requests, and documentation.
